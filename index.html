bootstrap-fullscreen-select

A custom select for @twitter bootstrap.
unlike others, this plugin provides a fullscreen select feature, that is trigger via a custom defined element.

Designed for Mobile UI, when it is not practical to use a dropdown select on a small display.

usage

html: 

<select class="form-control" id="theSelect"> // the ID of your select element
    <option value="0">Pizza</option>
    <option value="1">Burger</option>
    <option value="2">Tacos</option>
    <option value="3">Something else</option>
</select>

<button class="btn btn-default" id="theBtn">Show</button> //the ID of the element on which user will interact.

javascript:

IMPORTANT: please use ID to refer to an element. you can also use multiple initializations 

$(document).ready(function(){
    bootSelect.init({
        selectControl: '#theSelect', //the select element
        on: '#theBtn', //the element where user will interact
        title: 'What to eat?' //title of the full screen select.
    });
});

examples

select element:

HTML:

The select elements are shown for demonstration purpose, you can hide then in production.

<select class="form-control" id="mySelectElement"> //your select element
    <option value="0">Something</option>
    <option value="1">Something else</option>
    <option value="2">Something even else</option>
    <option value="3">Enough</option>
</select>

<button class="btn btn-default" id="myTriggerBtn">Show</button> //the element on which user will interact.

javascript: 

$(document).ready(function(){
    bootSelect.init({
        selectControl: '#mySelectElement', //the select element
        on: '#myTriggerBtn', //the element where user will interact
        title: 'Select something' //title of the full screen select.
    });
});

The plugin fired a change event on the select element on user interaction.

$('#mySelectElement').change(function(){
    alert($(this).val());
});
